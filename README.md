# u-project

## Dependencies
* MongoDb
* Loopback framework

You can configure db connection in file "server/datasources.json"

## Installation

Next

```
	$ npm install -g strongloop
	$ npm install
	$ cd client
	$ npm install
	$ cd ..
```

## Start

```
	$ node .
```
